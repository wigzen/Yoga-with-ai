About Our Project

Hey there! Yoga With AI , developed in 2021 by Group7.

We created Yoga With AI inspired by various Machine Learning models for visual recognition . This is my first Machine Learning model and our first attempt at Web Development. We have loved (almost) every second of this project and We have learnt more than we could have imagined.
Yoga With AI uses Pose.Net, developed by Google’s TensorFlow, and ml5.js, a JavaScript library for Machine Learning on the browser. Most of my learning was following The Coding Train’s YouTube playlist on Pose Estimation.
